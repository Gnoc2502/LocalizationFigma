Chọn từ danh sách hoặc nhập mã ISO (2-3 chữ cái, vd: pl, tr, uk)

💡 Prompt hoạt động thế nào?

Plugin trích text từ Figma → gửi kèm prompt cho AI → AI trả JSON đã dịch → plugin gắn lại vào Figma.

📥 Đầu vào (AI nhận được):

...Dịch sang Tiếng Việt (mã: vi)... Danh sách: ["Settings", "Delete", "Save changes"]

📤 Đầu ra (AI phải trả về):

{ "Settings": "Cài đặt", "Delete": "Xoá", "Save changes": "Lưu thay đổi" }

📌 3 biến tự động thay thế:

{{TEXT_LIST}} Danh sách text trích từ Figma (bắt buộc)
{{LANG_NAME}} Tên ngôn ngữ (vd: Tiếng Việt, Tiếng Nhật)
{{LANG_CODE}} Mã ISO ngôn ngữ (vd: vi, ja, ko)

✏️ Tuỳ chỉnh thêm vào prompt:

  • "Giữ nguyên tên riêng, brand name"
  • "Dùng giọng thân mật" hoặc "giọng trang trọng"
  • "Mỗi bản dịch tối đa 15 ký tự" (khi UI hẹp)
  • "Nếu là thuật ngữ kỹ thuật thì giữ nguyên tiếng Anh"

Phát hiện text dịch bị nhiều dòng hơn gốc, đề xuất dịch ngắn hơn
